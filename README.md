Project 1: Ordering Application 
The project includes the following core functionalities: 
• Load product data from a file (.txt) 
• Display the list of products  
• Allow users to select products and add them to the shopping cart 
(Note: users can add items multiple times)  
• View the shopping cart  
• Place an order (At this step, users must provide personal information and 
delivery address)  
• Display and confirm the order 
Order Details Must Include: 
o Order date (retrieved from the system’s current date)  
o Customer information  
o Delivery address  
o Expected delivery date  
o Selected products and corresponding quantities  
o Shipping fee  
o Total payment amount
